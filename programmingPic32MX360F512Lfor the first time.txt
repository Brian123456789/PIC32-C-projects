Programming The PIC32MX360F512L Using Explorer 16/32

Note: Your first time running, before things are set up, your programming icon will be greyed out. To get things ready for programming, do the following:

1. Plug in a usb cable to J18.
2. Right click on your project and make sure it is set as "main project"
3. Go to file, project properties. Set Family as PIC32 bit MCU's. In the MCU device window, put your part number here, in this case PIC32MX360F512L.
4. In the "Hardware tools" list, select PICKit3, apply, click ok.
5. Click "Build main Project" icon (a hammer)
6.Click "Make and program device main project" (an arrow into a chip icon)
